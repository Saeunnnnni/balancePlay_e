# BalancePlay 홈페이지 리뉴얼 프로젝트
<br/>

## 개요
- 사용자가 웹 사이트를 보다 쉽고 편리하게 이용할 수 있도록 직관적인 디자인
- 시각적으로 더 매력적이고 UI/UX를 구현(역동적인 홈페이지) 
- 반응형 웹 페이지 구현
